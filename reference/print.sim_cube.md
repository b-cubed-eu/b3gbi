# Print a Simulated Data Cube Object

Provides a summary representation of a sim_cube object, designed for
user-friendly display in the console.

## Usage

``` r
# S3 method for class 'sim_cube'
print(x, n = 10, ...)
```

## Arguments

- x:

  A sim_cube object.

- n:

  (Optional) Integer specifying the number of rows of cube data to
  display.

- ...:

  Additional arguments.
