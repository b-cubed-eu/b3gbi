# Print a Processed Data Cube Object

Provides a summary representation of a processed_cube_dsinfo object,
designed for user-friendly display in the console.

## Usage

``` r
# S3 method for class 'processed_cube_dsinfo'
print(x, n = 10, ...)
```

## Arguments

- x:

  A processed_cube_dsinfo object.

- n:

  (Optional) Integer specifying the number of rows of data to display.

- ...:

  Additional arguments.
