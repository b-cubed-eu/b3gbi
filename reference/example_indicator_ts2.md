# Time Series of Cumulative Species Richness for Bryophytes in South Africa

Example indicator containing a time series of cumulative species
richness for bryophyte occurrences in South Africa, 1875-2024
(occurrences from GBIF: 16.03.2024).

## Usage

``` r
example_indicator_ts2
```

## Format

An 'indicator_ts' object containing a tibble with 110 rows and 2
variables, as well as metadata

- year:

  a year the indicator was calculated for

- diversity_val:

  calculated richness value for the year
